Browser

Browser adalah perangkat lunak yang digunakan untuk mengakses dan menampilkan informasi di internet. Browser memungkinkan pengguna untuk menjelajahi situs web, menampilkan konten multimedia, dan berinteraksi dengan aplikasi berbasis web. Browser juga mendukung berbagai protokol dan teknologi web, seperti HTML, CSS, dan JavaScript.

Contoh Browser Populer:

  • Google Chrome:

    Browser yang cepat dan populer dengan dukungan ekstensi yang luas serta sinkronisasi antar perangkat.

  • Mozilla Firefox:

    Dikenal karena komitmennya terhadap privasi pengguna dan kemampuan kustomisasi melalui add-ons.

  • Microsoft Edge:

    Browser modern yang dikembangkan oleh Microsoft dengan integrasi fitur-fitur baru dan peningkatan performa.

Text Editor

Text editor adalah perangkat lunak yang digunakan untuk menulis dan mengedit teks dalam format sederhana tanpa format khusus seperti font atau ukuran huruf. Dalam pengembangan web dan pemrograman, text editor digunakan untuk menulis kode dalam berbagai bahasa pemrograman, termasuk HTML, CSS, JavaScript, Python, dan lainnya.
Beberapa fitur utama yang biasanya ada di text editor untuk pengembangan web adalah:

  • Highlighting Syntax:

    Menyoroti elemen-elemen dalam kode untuk memudahkan identifikasi dan debugging.

  • Auto-completion:

    Memberikan saran atau menyelesaikan kode secara otomatis untuk mempercepat penulisan kode.

  • Multi-language Support:

    Mendukung berbagai bahasa pemrograman.

  • Plugins & Extensions:

    Menambahkan fitur tambahan seperti debugging, integrasi dengan Git, atau live server untuk pratinjau langsung.

Contoh Text Editor Populer:

  • Visual Studio Code (VS Code):

    Text editor serbaguna dengan banyak fitur canggih, seperti integrasi langsung dengan Git dan ekstensi untuk berbagai bahasa pemrograman.

  • Sublime Text:

    Ringan dan cepat, cocok untuk pengguna yang membutuhkan performa tinggi.

  • Notepad++:

    Mudah digunakan dan sangat populer di kalangan pemula karena kesederhanaannya.

Langkah-Langkah Instalasi Text Editor

  • Mengunduh Text Editor

    • Ada beberapa pilihan text editor yang dapat diunduh. Peserta didik bisa memilih salah satu dari daftar berikut:

      • Visual Studio Code (VS Code):

        Text editor yang populer dengan banyak fitur, termasuk auto-completion, highlight syntax, dan integrasi dengan berbagai tools pengembangan web.

      • Sublime Text:

        Text editor ringan dan cepat, mendukung berbagai bahasa pemrograman.

      • Notepad++:

        Text editor yang sederhana dan mudah digunakan, cocok untuk pengguna pemula.s

  • Langkah Instalasi Visual Studio Code (VS Code)

    • Buka tautan berikut untuk mengunduh Visual Studio Code.

    • Pilih versi sesuai dengan sistem operasi yang digunakan (Windows, macOS, atau Linux).

    • Setelah file installer selesai diunduh, buka file tersebut untuk memulai proses instalasi.

    • Ikuti instruksi instalasi:

      • Pilih lokasi penyimpanan program.

      • Centang opsi untuk membuat shortcut di desktop agar mudah diakses.

      • Klik "Install" dan tunggu hingga proses selesai.

    • Setelah instalasi selesai, buka Visual Studio Code dengan mengklik shortcut di desktop atau melalui Start Menu.

  • Langkah Instalasi Sublime Text

    • Buka tautan berikut untuk mengunduh Sublime Text.

    • Pilih versi yang sesuai dengan sistem operasi (Windows, macOS, atau Linux).

    • Setelah file installer selesai diunduh, buka file tersebut untuk memulai proses instalasi.

    • Ikuti petunjuk instalasi sampai selesai.

    • Buka Sublime Text setelah instalasi selesai melalui desktop atau Start Menu.

  • Langkah Instalasi Notepad++

    • Buka tautan berikut untuk mengunduh Notepad++.

    • Pilih versi yang sesuai (biasanya ada pilihan 32-bit atau 64-bit untuk Windows).

    • Setelah file installer selesai diunduh, buka file tersebut untuk memulai instalasi.

    • Ikuti instruksi instalasi sampai selesai, termasuk pilihan untuk membuat shortcut di desktop.

    • Buka Notepad++ setelah instalasi selesai melalui shortcut di desktop atau Start Menu.
