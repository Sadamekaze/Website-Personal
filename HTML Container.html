HTML Container

HTML container adalah elemen yang digunakan untuk mengelompokkan atau membungkus elemen-elemen lain dalam sebuah halaman web. Fungsinya untuk mengatur tata letak, menyusun struktur, dan memberikan gaya pada grup elemen yang dibungkusnya. HTML container membantu pengembang web dalam mengorganisir konten agar lebih mudah dikelola, baik untuk tujuan visual maupun fungsional.

  • HTML dibentuk oleh elemen-elemen yang terdiri dari:

    • <div> (singkatan dari "division") adalah salah satu elemen paling umum dalam HTML yang digunakan sebagai container untuk membungkus elemen-elemen HTML lainnya. Elemen ini bersifat block-level, yang berarti akan menempati seluruh lebar layar dan dimulai pada baris baru.
    Fungsi utama dari <div> adalah untuk mengelompokkan elemen-elemen HTML lain (seperti teks, gambar, tabel, form, dll.) ke dalam satu wadah atau blok. Ini memudahkan pengaturan tata letak dan gaya tampilan menggunakan CSS. Dalam konteks tata letak, <div> sering digunakan sebagai container untuk membantu mengorganisir struktur halaman web.
    Contoh Penggunaan <div> sebagai Container:

    <!DOCTYPE html> <html> <style> div { background-color: #FFF4A3; } </style> <body> <h1>HTML DIV Example</h1> ​ <div> <h2>London</h2> <p>London is the capital of England</p> <p>London has over 13 million inhabitants</p> </div> <p>The yellow background is added to demonstrate the footprint of the DIV element.</p> </body> </html>

    Hasil pada browser:

    Dalam contoh ini, elemen heading <h1> dan dua elemen paragraf <p> dibungkus oleh <div>. Hal ini membuatnya lebih mudah untuk diatur tata letaknya secara keseluruhan dengan CSS.

  • HTML dibentuk oleh elemen-elemen yang terdiri dari:

    • Class adalah atribut yang digunakan untuk memberi nama pada satu atau lebih elemen HTML. Nama class digunakan untuk mengelompokkan beberapa elemen yang berbeda, sehingga memungkinkan kita untuk memberikan gaya atau tata letak yang sama kepada elemen-elemen tersebut.

    Dengan menggunakan class, elemen-elemen HTML yang dikelompokkan dalam sebuah div atau elemen lain dapat diatur secara seragam dalam desain dan tata letak.

    Kegunaan class:

    • Mengelompokkan elemen yang memiliki gaya atau fungsi yang sama.

    • Memudahkan penerapan gaya (CSS) pada beberapa elemen sekaligus.

    • Dapat digunakan di banyak elemen pada satu halaman HTML.

      • <!DOCTYPE html> <html> <head> <style> .city { background-color: tomato; color: white; border: 2px solid black; margin: 20px; padding: 20px; } </style> </head> <body> <div class="city"> <h2>London</h2> <p>London is the capital of England.</p> </div> ​ <div class="city"> <h2>Paris</h2> <p>Paris is the capital of France.</p> </div> ​ <div class="city"> <h2>Tokyo</h2> <p>Tokyo is the capital of Japan.</p> </div> </body> </html>

      Hasil pada browser:

      Dengan penggunaan class="section", kedua <div> tersebut memiliki gaya yang sama: latar belakang biru muda, padding 20px, dan margin antar blok 10px.

  • Penggunaan id dalam HTML

    • id adalah atribut yang digunakan untuk mengidentifikasi elemen HTML secara unik. Setiap elemen yang memiliki id harus memiliki nilai yang unik di seluruh halaman web, sehingga tidak boleh ada dua elemen yang memiliki id yang sama.

    Kegunaan id:

    • Memberikan identitas unik pada elemen tertentu.

    • Digunakan untuk mengatur satu elemen secara khusus dengan CSS atau memanipulasi elemen tersebut menggunakan JavaScript.

    • Tidak seperti class, id hanya bisa digunakan satu kali pada halaman yang sama.

      • <!DOCTYPE html> <html> <head> <style> .city { background-color: tomato; color: white; border: 2px solid black; margin: 20px; padding: 20px; } </style> </head> <body> <div class="city"> <h2>London</h2> <p>London is the capital of England.</p> </div> ​ <div class="city"> <h2>Paris</h2> <p>Paris is the capital of France.</p> </div> ​ <div class="city"> <h2>Tokyo</h2> <p>Tokyo is the capital of Japan.</p> </div> </body> </html>

      Hasil pada browser:

  • <div>, class, dan id Sebagai Container

    • Elemen <div>, bersama dengan class dan id, sering disebut sebagai container dalam konteks web development. Fungsi utamanya adalah membungkus elemen-elemen HTML agar bisa diatur dan dikelola secara lebih terstruktur. Elemen-elemen yang dibungkus di dalam <div> dapat dengan mudah diposisikan, dihias, dan disusun dengan CSS.

    Perbedaan Antara class dan id:

    • class digunakan untuk mengelompokkan beberapa elemen yang memiliki karakteristik yang sama. Anda bisa menggunakan satu class untuk banyak elemen.

    • id digunakan untuk mengidentifikasi satu elemen tertentu. Nilai id harus unik dalam satu halaman, sehingga hanya bisa digunakan sekali.

      • Fitur Class Id
      Penggunaan Untuk banyak elemen Untuk satu elemen
      Sintaks CSS .nama-class #nama-id
      Unik atau tidak? Tidak perlu unik, bisa digunakan berkali-kali Harus unik dalam satu halaman

      Contoh Penggunaan <div>, class, dan id sebagai Container:

      <!DOCTYPE html> <html> <head> <style> /* elemen dengan id "myHeader" */ #myHeader { background-color: lightblue; color: black; padding: 40px; text-align: center; } /* Style semua elemen dengan nama kelas "city" */ .city { background-color: tomato; color: white; padding: 10px; } </style> </head> <body> <h1 id="myHeader">My Cities</h1> <h2 class="city">London</h2> <p>London is the capital of England.</p> <h2 class="city">Paris</h2> <p>Paris is the capital of France.</p> <h2 class="city">Tokyo</h2> <p>Tokyo is the capital of Japan.</p> </body> </html>

      Hasil pada browser:

      Misalnya kita ingin membuat halaman web yang memiliki header, konten, dan footer. Berikut adalah cara membungkus elemen-elemen HTML tersebut dalam sebuah container <div>

  • Studi Kasus Penggunaan <div>, class, dan id dalam Halaman Web

    • Contoh: Misalkan kita sedang membuat halaman web untuk media pembelajaran interaktif, dan ingin membuat layout yang terdiri dari header, konten utama, dan footer.

    Berikut ini contoh implementasi:

    <!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Media Pembelajaran</title> <style> #header { background-color: darkblue; color: white; text-align: center; padding: 20px; } ​ .content { padding: 20px; } ​ .section { background-color: lightblue; padding: 10px; margin-bottom: 15px; } ​ #footer { background-color: lightgray; text-align: center; padding: 10px; } </style> </head> <body> <div id="header"> <h1>Media Pembelajaran Interaktif</h1> </div> ​ <div class="content"> <div class="section"> <h2>Materi 1: Pengantar HTML</h2> <p>Ini adalah materi pengantar tentang HTML.</p> </div> ​ <div class="section"> <h2>Materi 2L Dasar-dasar CSS</h2> <p>Ini adalah materi pengantar tentang CSS.</p> </div> </div> ​ <div id="footer"> <p>© 2024 Media Pembelajaran</p> </div> </body> </html>
