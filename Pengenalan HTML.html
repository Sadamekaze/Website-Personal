HTML

HTML (HyperText Markup Language) adalah bahasa standar yang digunakan untuk membuat dan merancang halaman web. HTML berfungsi untuk menyusun struktur konten pada halaman web, seperti teks, gambar, tautan, tabel, video, dan elemen interaktif lainnya. HTML bukanlah bahasa pemrograman, melainkan bahasa markup yang menggunakan tag untuk menandai elemen-elemen dalam sebuah dokumen web.

Memahami HTML

  • HTML dibentuk oleh elemen-elemen yang terdiri dari:

    • Tag pembuka: Menandai awal elemen. Misalnya, <p> menandai awal dari sebuah paragraf.

    • Konten: Teks atau elemen yang ditempatkan di antara tag pembuka dan tag penutup.

    • Tag penutup: Menandai akhir elemen. Misalnya, </p> menandai akhir dari sebuah paragraf.

      • Contoh elemen HTML:

      <p>Ini adalah paragraf.

  • Struktur Dasar HTML: Setiap halaman web dimulai dengan struktur HTML dasar.

      • Contoh struktur HTML sederhana:

      <html> <head> <title>Document </head> <body> <p>paragraf.</p> </body> </html>

Menjalankan File HTML di Browser

  • Membuat File HTML di Text Editor

    • Pertama, tulis kode HTML di text editor seperti Visual Studio Code (VS Code), Sublime Text, atau Notepad++.

    • Misalnya, Anda membuat kode HTML sederhana seperti berikut:

      • <html><!DOCTYPE html> <html lang="id"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Halaman HTML Pertama</title> </head> <body> <h1>Selamat Datang di Halaman Pertama Saya</h1> <p>Ini adalah contoh halaman HTML sederhana.<p> </body> </html>

  • Menyimpan File HTML

    • Setelah Anda selesai menulis kode HTML, simpan file tersebut:

      • Klik File > Save As atau tekan Ctrl + S (Windows) atau Cmd + S (Mac).

      • Beri nama file tersebut, misalnya index.html. Pastikan Anda menyimpan file dengan ekstensi .html.

      • Pilih lokasi di komputer Anda di mana Anda ingin menyimpan file tersebut, misalnya di folder Dokumen.

  • Menjalankan File HTML di Browser

    • Setelah menyimpan file HTML, ikuti langkah-langkah berikut untuk menjalankannya di browser:

    • Buka File di Browser Secara Langsung:

      • Buka File Explorer atau Finder (di Mac).

      • Navigasikan ke lokasi tempat Anda menyimpan file HTML tadi.

      • Klik dua kali (double-click) pada file index.html atau klik kanan dan pilih Open With, kemudian pilih browser seperti Google Chrome, Mozilla Firefox, atau Microsoft Edge.

      • File HTML akan terbuka di browser, dan Anda akan melihat halaman sesuai dengan kode HTML yang telah Anda tulis.

    • Buka File di Browser Secara Langsung:

      • Buka browser favorit Anda (Google Chrome, Firefox, Edge, atau lainnya).

      • Tekan tombol Ctrl + O (Windows) atau Cmd + O (Mac) untuk membuka file.

      • Cari file index.html yang sudah Anda simpan di komputer, kemudian klik Open.

      • File HTML akan ditampilkan di browser.

  • Pratinjau Perubahan di Browser

    • Setiap kali Anda membuat perubahan pada file HTML di text editor, simpan perubahan tersebut (tekan Ctrl + S atau Cmd + S).

    Setelah itu, kembali ke browser dan refresh/reload halaman (tekan Ctrl + R atau F5 di Windows, atau Cmd + R di Mac) untuk melihat perubahan yang baru saja Anda buat.

Fungsi tag HTML

  • Struktur dokumen:

    seperti yang dijelaskan oleh FreeCodeCamp, tag HTML membantu dalam menyusun struktur dokumen HTML. Mulai dari header sampai footer, setiap bagian dari halaman web dikelilingi oleh tag tertentu. Hal ini membantu browser dalam menafsirkan dan menampilkan konten secara tepat.

  • Format konten:

    tag HTML tidak hanya menyusun, tapi juga memformat konten. Misalnya, tag seperti <b> untuk tebal, <i> untuk miring, dan <u> untuk bergaris bawah membantu kita untuk menonjolkan bagian tertentu dari teks.

  • Menyisipkan media:

    tag HTML seperti <img> dan <video> memungkinkan kita untuk menyertakan gambar atau video dalam laman web. Fungsi ini membuat laman web menjadi lebih interaktif dan menarik.

  • Pembuatan link:

    tag HTML juga ada yang digunakan untuk membuat hyperlink, sehingga kamu dapat berpindah dari satu halaman ke halaman lain. Dengan fungsi ini, web menjadi dinamis dan saling terhubung.

  • Integrasi dengan CSS dan JavaScript:

    tag HTML juga berguna sebagai jembatan dengan CSS dan JavaScript. Kita bisa menambahkan gaya dan fungsi interaktif ke halaman web. Proses ini juga membantu dalam meningkatkan user experience.

  • Aksesibilitas:

    menurut TutorialsTonight, dengan menggunakan tag seperti <alt> pada gambar, kita dapat menjadikan laman web lebih mudah diakses oleh orang-orang yang memiliki keterbatasan.

  • Search Engine Optimization (SEO):

    tag HTML membantu mesin pencari seperti Google dalam mengenali dan menilai konten laman web, sehingga memungkinkan laman tersebut muncul dalam hasil pencarian yang relevan.

Tag HTML Dasar

Tag Keterangan
'<html>' Digunakan untuk mendefinisikan seluruh dokumen HTML.
'<head>' Berisi informasi meta tentang dokumen, seperti judul, link ke file CSS, dan lainnya.
'<title>' Dipakai di dalam elemen '<head>' untuk menetapkan judul halaman web yang akan ditampilkan pada tab browser. Penting untuk diingat bahwa dalam satu dokumen HTML, hanya akan diterima satu tag '<title>' saja.
'<body>' Merupakan bagian utama dari halaman web, di mana berisi semua konten yang ingin ditampilkan kepada audiens.
'<h1>'-'<h6>' Digunakan untuk judul dan subjudul, dengan '<h1>' sebagai judul utama dan '<h6>' sebagai judul terkecil.
'<p>' Berfungsi untuk paragraf teks.
'<a>' Digunakan untuk membuat hyperlink ke halaman web lain.
'<br>' Digunakan untuk membuat jeda baris dalam teks.
'<hr>' Dipakai untuk membuat garis horizontal. Biasanya sering digunakan sebagai pemisah konten.
'<div>' Merupakan tag untuk mengelompokkan elemen lainnya, sehingga memungkinkan kontrol yang lebih baik terhadap style dan layout. '<div>' adalah block-level element yang akan membuat baris baru saat dibuat.
'<span>' Tag untuk mengelompokkan elemen lainnya, tetapi sebagai inline element. Tidak akan membuat baris baru saat dibuat.

      <table>: Membuat tabel.

      <tr>: Mendefinisikan baris dalam tabel.

      <th>: Sel header tabel.

      <td>: Sel data tabel.

  • Elemen div

      • Memahami Elemen <div>

      Elemen <div> adalah elemen blok generik dalam HTML. Ini tidak memiliki makna semantik khusus, tetapi sering digunakan untuk mengatur tata letak dan gaya elemen lainnya.

      Menambahkan Warna dengan CSS

      Untuk memberikan warna pada elemen <div>, kita akan menggunakan properti background-color dalam CSS.

      <!DOCTYPE html> <html> <head> <title>Contoh Penggunaan Div dengan Warna</title> <style> .biru{ background-color: blue; padding: 20px; } .hijau{ background-color: green; text-align: center; } </style> </head> <body> <div class="biru"> <h1>Ini adalah bagian dengan latar belakang biru</h1> <p>Teks dalam div in akan memiliki latar belakang biru</p> </div> <div class="hijau"> <h1>Ini adalah bagian dengan latar belakang hijau</h1> <p>Teks dalam div in akan memiliki latar belakang hijau</p> </div> </body> </html>

      Penjelasan Kode

      • <style>: Tag ini digunakan untuk mendefinisikan gaya CSS di dalam dokumen HTML.

      • .biru dan .hijau: Ini adalah selector kelas CSS. Kita menggunakan kelas untuk memberikan gaya pada elemen yang sama.

      • background-color: Properti ini mengatur warna latar belakang elemen.

      • padding: Properti ini menambahkan jarak antara konten dan tepi elemen.

      • text-align: Properti ini mengatur penjajaran teks di dalam elemen

      Penjelasan Kode

      • Elemen <div>: Kita membuat dua elemen <div> dengan kelas biru dan hijau.

      • Selector Kelas: CSS akan mencari elemen dengan kelas yang sesuai (.biru atau .hijau).

      • Penerapan Gaya: Setelah ditemukan, CSS akan menerapkan gaya yang telah didefinisikan (warna latar belakang, padding, dan penjajaran teks).
